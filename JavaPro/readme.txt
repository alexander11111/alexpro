task1 - по внешней комманде(вводится из браузера ) производится вкл выкл устройства. 
Название устройства извлекается из бд и создается класс на котором выполняется toggle 
переключение состояния. Каждой комманде соотв свой класс с реализацией toggle переключения.
task2 - коммандная оболочка реализует комманды ls, pwd, ps pname="name"(возможность фильтрации процесса по pname) 
 Каждой комманде соотв свой класс с реализацией. Соответствие комманды и класса находится в xml файле( для отладки файл можно сгенер из кода).
